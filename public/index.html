New here? What you're looking at

SIDELINE is one small server with two doors onto the live World Cup — one for AI assistants, one for developers and agents — and it needs no API key from anyone. Here's the whole idea in four steps. Then press the button to watch it happen for real, on live data.

1

The wall

AI assistants are blind to the world's biggest live event, and every sports API demands a signup, a key, and a season-long subscription. An autonomous agent can't even fill in the form.

2

Free data, zero keys

Anyone — human or AI — reads live scores & fixtures for free. Devs call /matches and /live; AIs call the MCP tool get_live_scores. No account, no key.

3

The paywall is x402

Ask for a premium brief and you get back HTTP 402 with the exact price. No login screen — the payment itself is the authentication. That's the x402 standard.

4

Pay → unlock

Send the payment, get the data plus a reusable token. An AI can do this loop by itself — discover a match, hit the 402, pay, read the brief. That's the business model.

For developers — keyless HTTP

The payment is the auth. First call answers 402 with the price; pay, get the data:

curl https://THIS_HOST/matches          # free
curl https://THIS_HOST/premium/brief/ID # -> 402
curl -H "X-PAYMENT: <auth>" …/brief/ID  # -> 200

Run it yourself — 2 minutes

No keys, no signup — Node 18+ is all you need. Clone, install, start.

git clone https://github.com/emmatheo/sideline
cd sideline
npm install
npm start          # gateway + landing on :8791
npm run mcp        # MCP server on stdio (Claude Desktop / Cursor)
npm run smoke      # optional: PASS/FAIL the free → 402 → paid flow

Then open http://localhost:8791 and press ▶ Walk me through it. Full recording script in docs/demo-script.md.
